Setup

I know that in many schools, right clicking on the mouse or touch pad is disabled.

So, the only way of creating a batch file is through notepad. Most of the school have an application called: "Desktop Tools" or "Desktop Manager". You can go ahead an double click on that application, and then look for notepad. Once notepad is open.

Make your way to the File option at the top left corner.

Click Save As and name the file example.bat
That word "Example" which is underlined is where you can write your file name.. but be sure to remember the name. The .bat is important. It tells the computer that it is a Batch File.

Opening file

Previously we created a batch file, now lets cover opening. It is quite simple, you do the same steps as opening notepad. Then making your way to File option. Then to Open..., or click CTRL + O. That should then open the File Explorer.

The defualt thing for the File Explorer to do when it is opened from notepad is to look for files that end in .txt while our batch files end in .bat
The img down below shows that our File Explorer is only looking for files ending .txt
You can go ahead and click on that button that looks like

It should then display a list of option. Click on the one that reads All Files (*.*). Now the place that read Text Document (*.tx) should now read All Files (*.*) like the image below.

You should now be able to see your batch files, and also be able to open your batch files.

Run File

It is quite simple to run a batch file. All you have to do is Double Click on the file. Be conscientious because these Annoying Batch files could irritate you if they are run on your device.
